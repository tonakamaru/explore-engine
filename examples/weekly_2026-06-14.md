### 3. AWS Summit NY（6/17）: Kiro iOS アプリ・Bedrock AgentCore GA・AWS Transform　〔時期: 2026-06-17〕

**規模感**: 中 — クラウドサイドのエージェント基盤が「設定ベース GA」段階に到達

**能力**  
- **Kiro iOS アプリ（ゲーテッドプレビュー）**: Kiro セッションをスマートフォンから開始・監視・ステアリング・diff レビュー・変更承認が可能に。  
- **Bedrock AgentCore Harness（GA）**: モデル・ツール・スキル・指示をコンフィグで定義するだけで本番グレードのエージェントを数分でデプロイ可能。Managed Knowledge Base（Smart Parsing でマルチフォーマット RAG）と Web Search（ゼロデータエグレスで引用付き）が追加。

- **DevOps Agent**: リリース管理に自律テスト機能を追加。

**可能性空間**  
「エージェントを設定だけで本番投入できる」クラスが AWS エコシステム内で現実的になった。モバイルからのエージェント制御は非 PC シナリオのエージェント監視を開く。

**before → after**  
before: Bedrock AgentCore はプレビュー。エージェント構築には手動のオーケストレーションループ実装が必要。  
after: AgentCore Harness GA によりオーケストレーションループのコーディング不要化。Kiro はデスクトップ限定 → iOS からも操作可。

### 5. Google: Gemini CLI / Code Assist コンシューマー版サービス終了（6/18）→ Antigravity CLI 強制移行　〔時期: 2026-06-18〕

**規模感**: 中 — Google のコーディングツール系統が Gemini 名称から Antigravity ブランドに統合完了

**能力**  
AI Pro・AI Ultra・無料ティアのユーザー向け Gemini CLI と Gemini Code Assist IDE 拡張が 6/18 に終了。Antigravity CLI（Go 製バイナリ）への移行が要求される。Antigravity CLI は Antigravity 2.0 と同一バックエンドを使用し、マルチエージェント非同期ワークフロー・複数並列エージェントのターミナル非フリーズ動作を提供。Agent Skills・Hooks・サブエージェント・拡張機能（プラグインに改称）は引き継ぎ。ただし Google は「ローンチ時点で 1:1 の機能パリティはない」と明示。

**可能性空間**  
Google のコーディングエージェントが Gemini 系ツール群から Antigravity に一本化されたことで、将来の機能追加が単一スタックに集中する。

**before → after**  
before: Gemini CLI / Code Assist（消費者向け）が無料・有料ティアで提供。  
after: 6/18 以降はリクエストを受け付けない。Antigravity CLI（別バイナリ、機能差あり）への移行が必須。

### 7. Claude Code v2.1: 破壊的 git コマンドのブロック・/config 構文追加（6/18〜6/19）　〔時期: 2026-06-18〜2026-06-19〕

**規模感**: 小 — auto mode 安全性の漸進的改善

**能力**  
- v2.1.183（6/19）: auto mode で `git reset --hard`・`git checkout -- .`・`git clean -fd`・`git stash drop` を、ユーザーが「ローカル変更を捨てろ」と明示的に指示していない場合にブロック。  
- v2.1.181（6/18）: `/config key=value` 構文で対話・`-p`・Remote Control から任意設定を変更可能に（例: `/config thinking=false`）。macOS 向け `sandbox.allowAppleEvents` 設定追加。

**可能性空間**  
auto mode での意図しないコード損失リスクが低減。設定変更のためにファイルを直接編集する必要がなくなった。

**before → after**  
before: auto mode が破壊的 git コマンドを文脈無関係に実行する可能性があった。設定変更は settings.json の手動編集。  
after: 明示指示なしの破壊的 git 操作はブロック。プロンプトから `/config` で設定変更可。

**出典**  
- [Claude Code Updates by Anthropic - June 2026 | Releasebot](https://releasebot.io/updates/anthropic/claude-code)
