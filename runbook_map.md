# 単一ジャンル・天井フロンティア地図生成 — Runbook (map)

このプロンプトは1ジャンルの「現在の天井とそこに至るまでの主要な天井移動」を、長めの期間で網羅的に生成します。
週次デルタ用の `runbook.md` とは別の起動経路です（`mapgen.sh` が使う）。

`{{GENRE}}` `{{SCAN_FROM}}` `{{SCAN_TO}}` はラッパが実行時に置換します。

---

## ミッション

ジャンル「{{GENRE}}」について、**{{SCAN_FROM}} 〜 {{SCAN_TO}}** の期間で天井（能力フロンティア）を動かした主要な事象を Web リサーチで網羅し、現在のフロンティアの全体像を1本の markdown にまとめる。

## 出力プロトコル（厳守）

- **前置きや説明文を出さない**。最初のトークンから直接マーカー行で始める。
- 出力は以下の順で必ず含む:
  1. `<<<BEGIN_REPORT` を単独行で
  2. 本文 markdown
  3. `<<<END_REPORT` を単独行で
- 全部終わったら最後に `EXITCODE_OK` を単独行で出力して終了する。
- マーカー欠落・順序違反は失敗扱い（ラッパが再実行する場合がある）。

## 規約

- **規模感は 大→中→小 の順**に並べる
- スコープ外（{{SCAN_FROM}} より前 / {{SCAN_TO}} より後）の情報は入れない
- **個人への決め打ち禁止**（「君はXすべき」「あなたに向いてる」等）
- **製品 A vs B のピア比較禁止**。比較軸は**時間**だけ（before → after で固定ベースライン）
- **規模感の判定基準**: 「カテゴリ移動を起こしたか」「不可能 → 可能になったか」「桁違いに改善したか」。微改善は入れない
- **想定エントリ数**: 10〜15個（多すぎれば big rocks に絞る）
