# 週次デルタ収集（単一ジャンル） — Runbook

このプロンプトは1ジャンル分のリサーチを単一エージェントで完結させます。
複数ジャンルの並列化は `gather.sh` がシェル層で行うため、ここではサブエージェントを呼ばないでください（モデル非依存性のため）。

`{{GENRE}}` `{{DATE}}` `{{DATE_FROM}}` はラッパが実行時に置換します。

---

## ミッション

ジャンル「{{GENRE}}」について、**直近7日間（{{DATE_FROM}} 〜 {{DATE}}）**に天井（能力フロンティア）が動いた事象を Web リサーチで拾い、markdown レポートを出力する。

## 出力プロトコル（厳守）

- **前置きや説明文を出さない**。最初のトークンから直接マーカー行で始める。
- 出力は以下の3つのパートをこの順で必ず含む:
  1. `<<<BEGIN_REPORT` を単独行で
  2. 本文 markdown
  3. `<<<END_REPORT` を単独行で
- 全部終わったら最後に `EXITCODE_OK` を単独行で出力して終了する。
- これを満たさない出力は失敗扱いされる（マーカー欠落 → ラッパが再実行）。

## 本文フォーマット

```
# {{GENRE}} 週次デルタ {{DATE}}

直近7日間に天井を動かしたものだけ。動きが薄い場合は「動きなし」と1行だけ書いて終わる。

## 1. タイトル　〔時期: YYYY-MM-DD〕
**規模感**: 大/中/小 — 1行で理由
**能力**: 何が新たに可能になったか
**可能性空間**: 一般化したクラス（個人への決め打ちはしない）
**before → after**: 過去の天井 → 現在の天井
**出典**: https://...

## 2. ...
```

## 規約

- **規模感は 大→中→小 の順**に並べる
- **1週間より古い情報は除外**（{{DATE_FROM}} 以降のみ）
- **個人への決め打ち禁止**（「君はXすべき」「あなたに向いてる」等）
- **製品 A vs B のピア比較は禁止**。比較軸は**時間**だけ（before → after で固定ベースラインを書く）
- 動きが薄ければ無理に埋めず「動きなし」1行で終わる

## 完全形サンプル（few-shot・出力の型）

```
<<<BEGIN_REPORT
# AIコーディング 週次デルタ 2026-06-20

## 1. Anthropic Agent SDK の独立課金撤回　〔時期: 2026-06-15〕
**規模感**: 中 — エコシステム移行コストが当面ゼロに据え置き
**能力**: SDK経由のマルチエージェント実行が既存サブスクの範囲内で継続
**可能性空間**: マルチエージェント・オーケストレーション実装の障壁が当面維持される
**before → after**: SDK独立課金が予告されエコシステム側で移行検討が始まっていた → 当日撤回で従来通り
**出典**: https://www.anthropic.com/news/...

## 2. AWS Bedrock AgentCore GA　〔時期: 2026-06-17〕
**規模感**: 大 — マネージドエージェントランタイムの本番化
**能力**: 自前のランタイム構築なしにエージェント実行を AWS が引き受け
**可能性空間**: エージェント運用の責務がアプリ側からインフラ側へ移譲可能に
**before → after**: 各社が独自にエージェントランタイムを抱え込み → マネージド選択肢が GA
**出典**: https://aws.amazon.com/...

<<<END_REPORT
EXITCODE_OK
```

サンプルは形式の型を示すもの。実際のリサーチ結果は自分で取得すること。
